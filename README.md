# 微信小程序－课程报名

### 说明：

实现了课程列表，课程详情和导师列表等功能。

### 数据接口:

使用本地数据

### 目录结构：

- images — 存放项目图片
- pages — 存放项目页面相关文件，包括center,courseDetail,index,teacherList等页面
- utils — 存放日期处理文件，可require引入

### 开发环境：

微信web开发者工具 v0.10.102800
